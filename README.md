# youxuan
uni-app商城，优选商城

## 技术栈

- **uni-app** - 跨平台开发框架
- **Vue.js** - 渐进式JavaScript框架
- **Vuex** - 状态管理库
- **uView UI** - uni-app生态UI组件库
- **SCSS** - CSS预处理器
- **JavaScript** - 编程语言

## 功能特性
- ✅ 首页轮播展示
- ✅ 商品分类浏览
- ✅ 商品搜索功能
- ✅ 商品详情展示
- ✅ 购物车管理
- ✅ 订单管理（下单、查看订单）
- ✅ 地址管理（新增、编辑、删除）
- ✅ 优惠券管理
- ✅ 商品收藏
- ✅ 个人中心
- ✅ 用户登录/注册
- ✅ 消息中心
- ✅ 设置中心

## 项目结构

```
youxuanshop/
├── pages/          # 页面文件
│   ├── index/      # 首页
│   ├── category/   # 分类页
│   ├── cart/       # 购物车
│   ├── goods/      # 商品详情
│   └── ...
├── components/     # 组件文件
├── store/          # Vuex状态管理
├── common/         # 公共资源
├── static/         # 静态资源
├── uview-ui/       # UI组件库
├── App.vue         # 应用入口
├── main.js         # 主入口文件
├── manifest.json   # 应用配置
└── pages.json      # 页面配置
```
